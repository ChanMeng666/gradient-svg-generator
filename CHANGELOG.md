## [1.0.0]

### Added
- Initial documented release of Gradient SVG Generator.
- Node.js / Next.js service that generates animated gradient SVGs with customizable text overlays.
- Multiple gradient types and animation effects, plus templates for standard color schemes and pride flags.
- Deployable as an API endpoint for dynamic, animated text displays.
